# 🛡️ AMAAN – Personal Safety & Emergency Response Ap
**AMAAN** is a specialized safety application built with Flutter, designed to provide immediate assistance in critical situations. By leveraging hardware sensors and automated communication, it ensures help is dispatched even when manual phone operation is impossible.
## 🚀 Features
* **🎙️ Voice-Activated Alerts:** Trigger emergency protocols using a customizable panic keyword (e.g., "Help me").
* **📳 Shake Detection:** Uses the device's accelerometer to trigger alerts via physical motion.
* **📧 Automated Dispatch:** Instant SMS and Email notifications containing real-time location data.
* **🆘 Emergency Dashboard:** Quick-dial access to Police, Ambulance, and Women Protection services.

## 🛠️ Technical Stack
* **Framework:** Flutter
* **Language:** Dart & Java (Method Channels for sensors)
* **Email System:** Secure SMTP Integration
* **API:** Google Play Services (Location)
